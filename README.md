# Tasbeeh-Counter--Advanced-



📿 Tasbeeh Counter App

A simple and interactive Tasbeeh Counter App built using HTML, CSS, and JavaScript that helps users keep track of their tasbeeh counts digitally. The app features an easy-to-use interface with smooth button interactions, live counter updates, and a clean modern design for a better user experience.

✨ Features:

Digital tasbeeh counting system
Increase count with a single click
Reset counter functionality
Responsive and user-friendly UI
Smooth and clean design
Built with pure HTML, CSS & JavaScript

🚀 This project helped me strengthen my JavaScript skills by practicing event handling, DOM manipulation, counter logic, and responsive frontend design.
